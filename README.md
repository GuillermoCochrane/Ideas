# 🌟 Ideas - Recopilación de Recursos Web

> Una colección de ejemplos y recursos de desarrollo web para fácil acceso y aprendizaje

## 🚀 Descripción

Este proyecto reúne ejemplos y demos de técnicas de desarrollo web modernas que he ido encontrando y recopilando a lo largo del tiempo. Sirve como biblioteca personal de componentes reutilizables y soluciones probadas.

## 🛠 Tecnologías utilizadas

- **HTML5 semántico**
- **CSS3** (Grid, Flexbox, Animaciones, Transiciones)
- **JavaScript vanilla** para interactividad
- Técnicas modernas de UI/UX

## 📁 Estructura del Proyecto

```
ideas/
├── index.html          # Página principal con navegación
├── pages/              # Ejemplos categorizados
│   ├── scroll-animation/    # Animaciones con scroll
│   ├── sliders/             # Diferentes tipos de sliders
│   └── components/        # Componentes UI (details/summary, headers, etc)
└── assets/             # Recursos estáticos
    ├── css/
    └── images/
```

## 🌐 Uso

1. Clonar el repo: `git clone https://github.com/GuillermoCochrane/Ideas.git`
2. Abrir `index.html` en el navegador
3. Navegar por los ejemplos en cada sección
